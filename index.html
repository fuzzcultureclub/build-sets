How To Use 'Build' Sets

A quick reference guide to understanding build sets in your programming — what they mean, and how to execute them.

Scroll to explore
⬇️
⚒️ Type 1

Build to X reps @ Y RPE/RIR

Means ⤵️

Start with a familiar weight you know you can do in set 1, then slowly increase it for each set, until you hit a top weight in the last set that is programmed.

Examples ⤵️

"Build to 8 reps @ RPE 9 for Bench Press"
Build: Set 1 — 5kg × 8 Reps
Build: Set 2 — 7.5kg × 8 Reps
Build: Set 3 — 10kg × 8 Reps
Final Top Set: 15kg × 8 Reps
"Build to 8 reps @ 2 RIR for Back Squat"
Build: Set 1 — 30kg × 8
Build: Set 2 — 40kg × 8
Build: Set 3 — 50kg × 8
Build: Set 4 — 60kg × 8
Final Top Set: 70kg × 8
💡 If you get to that top set and realise you underestimated and feel like you can go heavier — feel free to add on another set if you have time until you reach the right intensity, OR make a note in the workout comments that next time you'll go heavier.
💨 Type 2

Build quickly then hit X reps @ Ykg

Means ⤵️

Before your working sets, ensure you build up the weight gradually so you aren't going into something heavy without a few warm-up sets (add these on yourself in Everfit and use as many sets as needed).

For reps, select something you can easily do that is a little higher than your working sets that slowly decreases until you hit the working weight that is programmed.

For warm-up sets we don't need to rest as much as they are easy working sets, so just rest as needed until you start your prescribed working sets.

Examples ⤵️

"Build quickly then hit 3 top sets of Bench Press @ 35kg"
Build Set 1: 20kg × 10
Build Set 2: 30kg × 8
Working Set 1: 35kg × 5
Working Set 2: 35kg × 5
Working Set 3: 35kg × 5
"Build quickly then hit 4 top sets of Deadlifts @ 110kg"
Build Set 1: 60kg × 8
Build Set 2: 80kg × 6
Build Set 3: 100kg × 4
Working Set 1: 110kg × 3
Working Set 2: 110kg × 3
Working Set 3: 110kg × 3
💡 As you get stronger, your starting weight for your warm-up sets will be heavier — similar to your warm-up movements.
🎯 Type 3

Build to a X RPE/RIR/RM(Rep Max) Then hit X sets of Y reps @ that weight

Means ⤵️

Use as many sets as you need to build to something specific. This might be a specific RPE / RIR or a 'rep max' (i.e. a 1RM, 3RM and so on).

Then once you've found that weight, do the programmed number of sets and reps at that specific weight.

For the build, use the amount of reps for the working sets. After each one, note the intensity — is that your 3RM? 8 RPE? 2 RIR? Or can you go heavier?

Rest as needed for the warm-up sets.

Examples ⤵️

"Build to 8 RPE for Leg Press then hit 5 sets of 5 reps at that weight"
Build Set 1: 70kg × 5
Build Set 2: 90kg × 5
Build Set 3: 110kg × 5
Build Set 4: 120kg × 5 ⬅️ RPE 7-8
Working Set 1: 125kg × 5
Working Set 2: 125kg × 5
Working Set 3: 125kg × 5
Working Set 4: 125kg × 5
Working Set 5: 125kg × 5
"Build to a 3RM Strict Press, then hit 4 sets of 3 reps at that weight"
Build Set 1: 15kg × 3
Build Set 2: 17.5kg × 3
Build Set 3: 20kg × 3
Build Set 4: 22.5kg × 3 ⬅️ Close to failure
Working Set 1: 25kg × 3
Working Set 2: 25kg × 3
Working Set 3: 25kg × 3
💡 If you do a lift that is CLOSE to the right intensity, you don't have to do another build set — you can probably assume your working sets will be slightly heavier and go straight in.
🥵 Type 4

Build to a XRM

Means ⤵️

Use as many sets as you need to build to a max amount of reps — i.e. 3RM, 1RM, 8RM and so on.

For reps: choose a starting point a little higher than what is programmed so you have time to warm up your lifts — but make sure you don't do too much and fatigue yourself before the top set.

Rest as you need to between each set.

Sometimes I might put a time constraint on this — sometimes not. If you're a person that likes to be in and out of the gym quickly I might give you a time limit so you don't spend too long on it.

Examples ⤵️

"Build to a 1RM Back Squat"
Build Set 1: 30kg × 5
Build Set 2: 40kg × 3
Build Set 3: 50kg × 3
Build Set 4: 60kg × 2
Build Set 5: 75kg × 1
Build Set 6: 77.5kg × 1
Build Set 7: 80kg × 1 ⬅️ 1RM
Final Set 8: 82.5kg × 0 ⬅️ Failed
"Take 10 minutes to find a 2RM Split Squat"
Build Set 1: 20kg × 4/4
Build Set 2: 25kg × 3/3
Build Set 3: 30kg × 2/2
Build Set 4: 32.5kg × 2/2
Build Set 5: 35kg × 2/2 🏆
💡 If you want to max out, you may end up trying a weight that you fail — which is 100% normal. We don't know where our ceiling is if we don't try for the big lifts!
💡 Sometimes you'll hit a weight and feel like 'yup, I don't have any more' — so you may choose to leave it there without trying to push for another one. It depends how you feel on the day!
🔙 Type 5

Build to X RPE/RIR/RM(Rep Max) Then hit 1 back off set for max reps @ Y% of the top set's weight

Means ⤵️

Build up to a specific number of reps at a programmed intensity, or build to a rep max, then with the top weight, hit a back-off set at a percentage of that weight — and usually we take that all the way to failure.

This allows us to get more volume in at a lighter weight after we hit some heavy sets — therefore allowing us to progressively overload a movement more. Each week we can further that by using the back-off set to chase an extra rep or two.

We can also use back-off sets at the end of slow tempo work — to hit our fast-twitch muscle fibres after something slow and heavy.

Examples ⤵️

"Build to RPE 9 @ 4 reps, then hit a max back off set @ 95% of your top set weight."
Build Set 1: 40kg × 4
Build Set 2: 50kg × 4
Build Set 3: 60kg × 4
Build Set 4: 65kg × 4 ⬅️ RPE 9
Back Off Set: 62kg × Failure ⬅️ 95%
"Hit 4×5 @ 50kg, Tempo 3211, then hit a max back off set @ 80% Tempo 20×1"
Set 1: 50kg × 5
Set 2: 50kg × 5
Set 3: 50kg × 5
Set 4: 50kg × 5
Back Off Set: 40kg × Failure ⬅️ 80%
💡 To calculate a percentage, just do the percentage as a decimal multiplied by the weight. Example: 0.95 × 65kg = 61.75kg → round up to 62kg.

And that's how you use build sets like a champ 💜
